Simple ATM [GUI Based]

Description

We teamed work to create a simple ATM as a group project. This project's aims to create auto-teller machine alike application. This involves user registration, login, balance inquiry, withdraw, deposit, and change password.

Purpose

User friendly auto-teller machine application which users are allow to register, login, check balance inquiry, withdraw, deposit, and change password.

Users' Benefit

Easy banking and hassle-free access to bank account.

Features

  • Check your account balance anytime, anywhere.
  • Withdraw cash from ATMs.
  • Deposit money into your account.
